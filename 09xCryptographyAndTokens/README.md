# Cryptography and Tokens

Cryptography and Tokens are very important aspects of programming when
it comes to ensuring that applications are secure against any intrusion that
might put user data at risk.

In this repository we will be exploring those concepts that have to do with
the listed topic, such as hashing, hashing algorithms, tokens, secret keys
and etc.

## Requirements
See the requirements.txt script for all the required dependencies for this
repo.

Or simpley just clone the repo and run:
`pip install -r requirements.txt`
