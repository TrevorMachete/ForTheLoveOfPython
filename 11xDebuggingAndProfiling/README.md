# Debugging and Profiling

This repo instriduces us to one of the most important tasks of a programmer - 
debugging and profiling. As you might know, debugging takes up the most part
of a programmer's time because no matter how well we write our code, bugs
will always creep in and sometimes finding them and rooting them out may take
more time and effort we desire, so it is important that every programmer
get well vested in matters relating to debugging code.

## Repo files
In this repo you will find the following files:

 - log_to_file.py  
 This script introduces us to configuring logging to output to a file.

 ### Instructions

 To run all or any of the scripts, clone the repo to you local environment,
 cd into the parent directory of the modules in your console and run command:
 `python3 [module_name.py]`

 Remember that the square brackets are not part of the command syntax.
